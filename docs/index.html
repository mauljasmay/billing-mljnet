Integrated ISP Management System

Sistem manajemen ISP terlengkap dengan integrasi WhatsApp Gateway untuk mengelola billing, layanan pelanggan, dan operasi jaringan secara efisien.

Mulai Sekarang Lihat di GitHub

Fitur Utama

Solusi komprehensif untuk manajemen penyedia layanan internet

WhatsApp Gateway

Interaksi pelanggan, pengiriman voucher, pelaporan gangguan, dan notifikasi otomatis melalui WhatsApp.

Integrasi GenieACS

Manajemen CPE (Customer Premises Equipment) yang terpusat dan otomatis.

Sistem Billing

Pembuatan invoice otomatis, pelacakan pembayaran, dan remittance yang terintegrasi.

Manajemen Agen & Teknisi

Peran, kontrol akses, dan penugasan pekerjaan yang fleksibel dan terstruktur.

Migrasi Database

Pembaruan skema berbasis SQL untuk pengembangan yang berkelanjutan dan aman.

Pemetaan Jaringan

Manajemen ODP, tiang, dan tata letak kabel secara visual dan terintegrasi.

Instalasi Cepat

Mulai gunakan Gembok Bill dalam hitungan menit

# 1. Clone repository
git clone https://github.com/alijayanet/gembok-bill.git
cd gembok-bill

# 2. Instal dependensi
npm install

# 3. Konfigurasi environment
cp .env.example .env
# Edit file .env dengan konfigurasi Anda

# 4. Inisialisasi database
npm run setup

# 5. Jalankan aplikasi
npm start
